# 3강 Java 기본문법2, 클래스와 상속1

## 배열

### 배열

- 같은 자료형의 원소를 정해진 개수만큼 가지고 있는 객체
  - 배열의 크기는 배열이 초기화 또는 생성될 때 정해 짐
  - 숫자 인덱스를 사용하여 특정 원소를 다룸

### 배열의 초기화

- 선언과 동시에 중괄호를 이용하여 초기값을 지정
  - 자동으로 메모리 공간이 확보됨
  - 초기화 또는 생성 과정을 거쳐야 배열의 원소를 사용할 수 있음

### 배열의 생성

- 배열의 원소가 사용할 메모리 공간의 생성
  - new 연산자를 이용

### 배열의 크기

- 배열은 크기를 가지는 내장 속성 length를 가짐
- 사용법은 배열이름.length

## 문자열

### String 클래스

- String 클래스는 문자열을 표현하고 처리하기 위한 참조형
- String 형의 변수는 참조형이나 기본형 변수처럼 사용할 수 있음

### 문자열의 +연산자

- 두 문자열을 연결하는 것
- (문자열 + 기본형) 또는 (문자열 + 다른 참조형)도 가능
- print()나 printLn()에서 자주 사용됨

## Scanner 클래스

### Scanner 클래스

- 키보드나 파일로부터 다양한 자료를 입력 받을 때 사용
  - 기본적으로 공백 문자로 구분되는 단어 단위로 입력됨
  - 문자열이나 기본형 값의 입력을 위해 next() 또는 nextInt()와 같은
    메소드를 제공함
- 키보드에서 입력을 받는 Scanner 객체 만들기
  - System.in을 이용하여 Scanner 객체를 만들고 사용함

## 클래스 정의

### 클래스 정의

- 동종 객체를 표현하기 위한 데이터 필드와 메소드를 정의함
  - 부모 클래스의 상속, 인터페이스의 구현, 추상 클래스, final 클래스 등을 정함
